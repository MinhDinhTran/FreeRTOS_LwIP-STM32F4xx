<h3>LwIP work with RTOS in STM32F4xx</h3>
<p>This is a code framework that base on STM32F4xx MCU. It combines FreeRTOS with LwIP, to make it more convenient for users to develop ether-app on STM32F4 series MCU.</p>
<p>这是一个基于STM32F4xx系列的代码框架，在FreeRTOS上运行LwIP，目的是为了之后的相关开发提供方便的平台。</p>
